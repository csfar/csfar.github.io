Formworks

A framework inspired by Server-driven UI for building forms from JSON.

GitHub

Formworks is available as a Swift Package. It makes use of View Code to create each component as a UITableViewCell. The JSON decoding is done through a custom decoder made with default values in mind. Every form can be customized through the JSON. The entire development process took advantage of a CI pipeline (GitHub Actions) for running tests made with XCTest, running SwiftLint and for generating the documentation using Jazzy.

Team: Rafael Galdino, Edgar Sgroi, Victor Falcetta and Cassia Barbosa

Memora

Memora is an iOS app for recording, collecting and remembering audio memories.

GitHub - App Store

Memora is developed following a MVVM-like architecture with some UIViewController containment, all done using View Code. It uses Core Data as its persistence layer with NSFetchedResultsController for management of data to be displayed. Recording and playing of memories is implemented using AVFoundation, with animations done using Core Animation. Lastly, logging is done through OSLog.

Coremelysis

Coremelysis is an app for experience sampling that uses machine learning for sentiment analysis.

GitHub

Coremelysis architecture is similar to MVVM with some UIViewController containment, all done using View Code. It makes use of the native punatural language library, Natural Language, to peform Sentiment Analysis on text. Each analysis is saved using Core Data and displayed in a history using NSFetchedResultsController. The app provides an option to choose between to ML models, saving the chosen option in the UserDefaults. Inferences can be made using Siri Shortcuts.

Team: Rafael Galdino

Blink

A SwiftUI app made for brainstorming using an Apple TV and multiple iPhones.

GitHub - App Store

Both iOS and tvOS versions of the app are built using SwiftUI and Multipeer Connectivity. Some UIViewRepresentable was used to display the Multipeer Connectivity available devices screen. OSLog was used for debugging purposes.

Team: Edgar Sgroi and Victor Falcetta
