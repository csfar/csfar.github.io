Memora

Memora is an iOS app for recording, collecting and remembering audio memories.

GitHub - App Store

Memora is developed following a MVVM-like architecture with some UIViewController containment. It uses Core Data as its persistence layer with NSFetchedResultsController for management of data to be displayed. Recording and playing of memories is implemented using AVFoundation, with animations done using Core Animation. Lastly, logging is done through OSLog.

Coremelysis

Coremelysis is an app for experience sampling that uses machine learning for sentiment analysis.

GitHub

Coremelysis architecture is similar to MVVM with some UIViewController containment. It makes use of the native punatural language library, Natural Language, to peform Sentiment Analyis on text. Each analysis is saved using Core Data and displayed in a history using NSFetchedResultsController. The app provides an option to choose between to ML models, saving the chosen option in the UserDefaults. Inferences can be made using Siri Shortcuts.

Team: Rafael Galdino

Blink

A SwiftUI app made for brainstorming using an Apple TV and multiple iPhones.

GitHub - App Store

Both iOS and tvOS versions of the app are built using SwiftUI and Multipeer Connectivity. Some UIViewRepresentable was used to display the necessary Multipeer Connectivity devices screen. OSLog was used as a debugging tool.

Team: Edgar Sgroi, Victor Falcetta
